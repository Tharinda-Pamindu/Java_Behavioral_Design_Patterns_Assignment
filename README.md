# Java Behavioral Design Patterns Assignment

This repository contains implementations of several common behavioral design patterns in Java. Each project is self-contained in its own directory and demonstrates a specific pattern.

## Projects

Below is a list of the design patterns implemented in this assignment. Click on a link to navigate to the specific project directory.

-   **[Observer Pattern](./ObserverDesign/)**: A Weather Station that automatically notifies multiple display devices when its data changes.

-   **[Strategy Pattern](./StrategyDesign/)**: A Shopping Cart that allows the payment method (e.g., Credit Card, PayPal) to be selected and changed at runtime.

-   **[Command Pattern](./CommandDesign/)**: A smart home remote control that encapsulates requests (e.g., "turn light on") as objects, enabling features like undo.
